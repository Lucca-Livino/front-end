# 📚 Estudos de Frontend

Repositório dedicado a estudos e projetos práticos de desenvolvimento frontend, cobrindo desde HTML/CSS puro até aplicações modernas com React e Vite.


---

## 🛠️ Tecnologias Utilizadas

- **HTML5 / CSS** — estrutura e estilização de páginas estáticas
- **JavaScript** — lógica e interatividade no lado do cliente
- **React** — biblioteca para construção de interfaces de usuário
- **Vite** — ferramenta de build rápida para projetos modernos

---

## 🚀 Como rodar os projetos

### Projetos com HTML puro

Basta abrir o arquivo `index.html` diretamente no navegador, ou usar uma extensão como o **Live Server** no VS Code.

### Projetos com Vite + React

```bash
# Acesse a pasta do projeto
cd nome-do-projeto

# Instale as dependências
npm install

# Rode o servidor de desenvolvimento
npm run dev
```
